PaintPort
by

Bemalte Bambu-/MakerWorld-Modelle mit den richtigen Farben auf deinem Drucker — Ziel: PrusaSlicer (Core One INDX), Bambu Studio oder Snapmaker Orca, inkl. ColorMix-Mischfarben. Komplett lokal im Browser.

Quickstart

  1. Bemalte 3MF (Bambu Studio / MakerWorld / OrcaSlicer) in das Feld oben ziehen.
  2. Eintragen, welche Spulenfarbe in welchem Drucker-Slot steckt (oder ein Preset wählen).
  3. „Auto-Zuordnung" klicken und das Ergebnis prüfen — fehlende Farben werden optional als ColorMix gemischt.
  4. Ziel-Slicer wählen, exportieren und die Datei dort als Projekt öffnen.
Wichtig: Die exportierte 3MF im Ziel-Slicer als Projekt öffnen (Datei → Öffnen bzw. beim Drag&Drop „Projekt" wählen). Sonst gehen ColorMix-Mischfarben und Slot-Farben verloren — das Painting selbst bleibt erhalten.

Nutzung & Datenschutz

Deine Daten

Alles passiert in deinem Browser: keine Uploads, kein Server, kein Tracking, keine Cookies. Gespeichert werden nur zwei lokale Einstellungen (Sprache, „Willkommen gesehen") im localStorage deines Browsers.

Open Source

PaintPort ist freie Software unter der AGPL-3.0 — Quellcode auf GitHub. © Nils Stackler / perspektive3D.

Keine Gewähr

PaintPort wird ohne Gewährleistung bereitgestellt. Druckergebnisse (insbesondere ColorMix-Farbtreue) können abweichen — prüfe die Zuordnung vor dem Druck.
